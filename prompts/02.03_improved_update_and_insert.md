Look at the  @AddEditViewModel.kt . In the init function I want you to determine if this is a new or an update  scenario. Then set a local private value `isNew` to true if it is a new mood. Then set another local variable moodId that is non-nullable to 0 if it is a new mood or set it to the entryId if it is an edit. When the mood is saved use the moodId and the flag to either update or insert. moodId will then always have the correct value.
