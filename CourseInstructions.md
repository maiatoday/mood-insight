# Overview
This course is intended to showcase Gemini in Android Studio features by building a complete app. Each step is described in an issue in this repository. The intention is that you follow the steps sequentially but if there is a section that is particularly interesting, checkout the starting branch for the step and do the steps out of order.

The app is a small mood tracking app. 

# How
1. Fork the repo so you have your own copy
2. Open the issue for the topic that you are interested in and check out the starting branch
3. Clear the agent context
4. Read the instructions of the issue and follow the steps. There are prompts for the steps saved with the project. You can retrieve the prompt with the `@` command in the Agent tab of the Gemini in Android Studio feature. You can also modify the prompt before running it.
5. If the prompts aren't showing up, find all the prompts in the [prompts folder](prompts)

# Tips

* Play around with the code and the prompts. If you get yourself into a bad state, just revert the code and start again.
* Remember to clear the context before starting each step
* Don't close the issues as they are used to guide the course
* If you run out of tokens on the free tier, you can [generate an API key](https://developer.android.com/studio/gemini/add-api-key) for free. 

# How was this app created

The app was created with the AI supported feature that allows you to give a prompt to create a basic app. See the .agent/plan.md file
